Описание всех используемых решателей

// Решатели COCG
"COCG"
 - обычный COCG
 - решатель с БСУ-1, вроде рабочий
"COCG_Di"
 - COCG с диагональным предобуславливанием
 - решатель с БСУ-1, вроде рабочий
"COCG_GS"
 - COCG с предобуславливанием Гаусса-Зейделя
 - решатель с БСУ-1, вроде рабочий
"COCG_LLT"
 - COCG с предобуславливанием LL^T
 - решатель с БСУ-1, вроде рабочий
"COCG_LDLT"
 - COCG с предобуславливанием LDL^T
 - решатель с БСУ-1, вроде рабочий
"COCG_Smooth"
 - обычный COCG со сглаживанием невязки
 - модифицированный решатель с БСУ-1, вроде рабочий
"COCG_Di_Smooth"
 - COCG со сглаживанием невязки и диагональным предобуславливанием
 - модифицированный решатель с БСУ-1, вроде рабочий
"COCG_GS_Smooth"
 - COCG со сглаживанием невязки и предобуславливанием Гаусса-Зейделя
 - модифицированный решатель с БСУ-1, вроде рабочий
"COCG_LLT_Smooth"
 - COCG со сглаживанием невязки и предобуславливанием LL^T
 - модифицированный решатель с БСУ-1, 100% рабочий
"COCG_LDLT_Smooth"
 - COCG со сглаживанием невязки и предобуславливанием LDL^T
 - модифицированный решатель с БСУ-1, вроде рабочий

// Решатели COCG с OpenMP
"COCG_Smooth_OpenMP"
 - COCG со сглаживанием невязки, распараллеленный через OpenMP
 - модифицированный решатель с БСУ-2, вроде рабочий
"COCG_Di_Smooth_OpenMP"
 - COCG со сглаживанием невязки и диагональным предобуславливанием, распараллеленный через OpenMP
 - модифицированный решатель с БСУ-2, вроде рабочий

// Решатели COCG с MKL
"COCG_Smooth_MKL"
 - обычный COCG со сглаживанием невязки, использующий библиотеку MKL
 - модифицированный решатель с БСУ-2, вроде рабочий
"COCG_Di_Smooth_MKL"
 - COCG со сглаживанием невязки и диагональным предобуславливанием, использующий библиотеку MKL
 - модифицированный решатель с БСУ-2, вроде рабочий
"COCG_LLT_Smooth_MKL"
 - COCG со сглаживанием невязки и предобуславливанием LL^T, использующий библиотеку MKL
 - модифицированный решатель с БСУ-2, 100% рабочий

// Решатели GMRES
"GMRES_Complex"
 - обычный GMRES
 - решатель с БСУ-1, вроде рабочий
"GMRES_Complex_Di"
 - GMRES с диагональным предобуславливанием
 - решатель с БСУ-1, вроде рабочий
"GMRES_Complex_LLT"
 - GMRES с предобуславливанием LL^T
 - решатель с БСУ-1, вроде рабочий
"GMRES_Complex_LDLT"
 - GMRES с предобуславливанием LDL^T
 - решатель с БСУ-1, вроде рабочий

// Решатели GMRES с OpenMP
"GMRES_Complex_OpenMP"
 - обычный GMRES, распараллеленный через OpenMP
 - модифицированный распараллеленный решатель с БСУ-1, вроде рабочий
"GMRES_Complex_Di_OpenMP"
 - GMRES с диагональным предобуславливанием, распараллеленный через OpenMP
 - модифицированный распараллеленный решатель с БСУ-1, вроде рабочий

// Решатели GMRES с MKL
"GMRES_Complex_MKL"
 - обычный GMRES, использующий библиотеку MKL
 - модифицированный решатель с БСУ-1, переделанный под MKL, вроде рабочий
"GMRES_Complex_Di_MKL"
 - GMRES с диагональным предобуславливанием, использующий библиотеку MKL
 - модифицированный решатель с БСУ-1, переделанный под MKL, вроде рабочий
"GMRES_Complex_LLT_MKL"
 - GMRES с предобуславливанием LL^T, использующий библиотеку MKL
 - модифицированный решатель с БСУ-1, переделанный под MKL, вроде рабочий

// Решатели BiCG
"BiCG_Complex_Smooth"
 - обычный BiCG со сглаживанием невязки
 - оптимизированный решатель Макса, вроде рабочий, но медленный

// Решатели BiCGStab"
"BiCGStab_Complex_Smooth" - обычный BiCGStab со сглаживанием невязки
