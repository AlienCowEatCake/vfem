Описание типа решателя состоит из двух частей: название собственно решателя и название
предобуславливателя. Общий синтаксис для описания решателей:

"SolverName<PreconditionerName>"

Если предобуславливатель не указан, то используется пустой предобуславливатель.

Описание всех используемых решателей:

// Решатели COCG
"COCG"
 - COCG
 - решатель, изначально писавшийся Максом, модифицированный во время БСУ-1, рабочий
"COCG_Smooth"
 - COCG со сглаживанием невязки
 - модифицированный COCG, решатель с БСУ-1, рабочий
"COCG_Smooth_OpenMP"
 - COCG со сглаживанием невязки, распараллеленный через OpenMP
 - модифицированный COCG_Smooth, решатель с БСУ-2, рабочий
"COCG_Smooth_MKL"
 - COCG со сглаживанием невязки, использующий библиотеку MKL
 - модифицированный COCG_Smooth, решатель с БСУ-2, рабочий

// Решатели COCR
"COCR"
 - COCR
 - решатель с БСУ-1, вроде рабочий
"COCR_Smooth"
 - COCR со сглавживанием невязки
 - модифицированный COCR, решатель с БСУ-1, вроде рабочий

// Решатели BiCG
"BiCG_Complex"
 - Комплексный BiCG
 - старый решатель Макса, вроде рабочий, но медленный
"BiCG_Complex_Smooth"
 - Комплексный BiCG со сглаживанием невязки
 - Модифицированный BiCG_Complex, вроде рабочий, но медленный

// Решатели BiCGStab
"BiCGStab_Complex"
 - Комплексный BiCGStab
 - старый решатель Макса, работает нестабильно
"BiCGStab_Complex_Smooth"
 - Комплексный BiCGStab со сглаживанием невязки
 - Модифицированный BiCGStab_Complex, работает нестабильно

// Решатели GMRES
"GMRES_Complex"
 - Комплексный GMRES
 - решатель с БСУ-1, рабочий
"GMRES_Complex_OpenMP"
 - Комплексный GMRES, распараллеленный через OpenMP
 - модифицированный GMRES_Complex, рабочий
"GMRES_Complex_MKL"
 - Комплексный GMRES, использующий библиотеку MKL
 - модифицированный GMRES_Complex, рабочий


Описание всех используемых предобуславливателей

// Пустые предобуславливатели
"Nothing"
 - стандартный пустой предобуславливатель
"Nothing_OpenMP"
 - пустой предобуславливатель, распараллеленный через OpenMP
"Nothing_MKL"
 - пустой предобуславливатель, использующий библиотеку MKL

// Диагональные предобуславливатели, A=S*Q : S=Di, Q=I
"Di"
 - стандартный диагональный предобуславливатель
"Di_OpenMP"
 - диагональный предобуславливатель, распараллеленный через OpenMP
"Di_MKL"
 - диагональный предобуславливатель, использующий библиотеку MKL

// Предобуславливатели неполным LL^T разложением, A=S*Q : S=L, Q=L^T
"LLT"
 - стандартный предобуславливатель неполным LL^T разложением
"LLT_MKL"
 - предобуславливатель неполным LL^T разложением, использующий библиотеку MKL

// Предобуславливатели неполным LDL^T разложением, A=S*Q : S=L*D, Q=L^T
"LDLT"
 - стандартный предобуславливатель неполным LDL^T разложением

// Предобуславливатели типа Гаусса-Зейделя, A=S*Q : S=(D+L)^-1*D*(D+U)^-1, Q=I
"GS"
 - стандартный предобуславливатель типа Гаусса-Зейделя
